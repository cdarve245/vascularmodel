What type of image data is contained in the repository?

Each project contains medical image data in .vti format.

How do I use this website?

The project files can be accessed from the Repository. Each model can be downloaded by clicking on the model name appearing when hovering on the corresponding picture. The projects are compatible with SimVascular; please refer to the Documentation for more information on how to open and use them.
